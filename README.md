# SQL Business Insights Project

## Overview
This project analyzes customer orders, products, and sales data using SQL to generate meaningful business insights.

## Dataset
- Customers
- Products
- Orders

## Key Insights
- Revenue by product and category
- Monthly sales trends
- Top-selling products
- Repeat customers

## Skills Used
- SQL Joins
- Aggregations
- Group By & Having
- Business Analysis

## Tools
- SQL (MySQL / PostgreSQL / SQLite)

It is created as a demo portfolio project to showcase relevant skills.
